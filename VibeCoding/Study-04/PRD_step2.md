# PRD Step 2 — 레시피 생성

## 개요

Step 1에서 인식된 재료 목록을 바탕으로 OpenRouter API(`nvidia/nemotron-3-super-120b-a12b:free`)를 호출하여 맞춤 레시피를 생성하고, 결과를 사용자에게 보기 좋게 제공한다.

---

## 목표

- 인식된 재료로 만들 수 있는 레시피를 AI가 자동 생성
- 사용자 취향(인원수, 요리 시간, 식단 제한)을 반영한 필터 옵션 제공
- 생성된 레시피를 Step 3의 프로필에 저장할 수 있도록 연결

---

## 사용자 스토리

- 사용자는 Step 1에서 넘어온 재료 목록을 확인하고 레시피 옵션을 설정할 수 있다.
- "레시피 생성하기" 버튼을 누르면 AI가 여러 레시피를 추천한다.
- 각 레시피는 요리명, 재료 목록, 단계별 조리법, 예상 시간, 난이도를 포함한다.
- 사용자는 마음에 드는 레시피를 "저장하기" 버튼으로 프로필에 저장할 수 있다.
- 결과가 마음에 들지 않으면 "다시 생성하기"로 새 레시피를 요청할 수 있다.

---

## 기능 요구사항

### F1. 레시피 옵션 설정
| ID | 요구사항 |
|----|----------|
| F1-1 | 인원수 선택: 1인 / 2인 / 4인 이상 |
| F1-2 | 요리 시간 선택: 15분 이내 / 30분 이내 / 1시간 이내 / 무제한 |
| F1-3 | 식단 필터: 일반 / 채식 / 저칼로리 / 글루텐 프리 (다중 선택) |
| F1-4 | 추천 개수: 1개 / 3개 / 5개 |
| F1-5 | Step 1 재료 목록을 읽기 전용으로 표시, 뒤로가기로 수정 가능 |

### F2. AI 레시피 생성
| ID | 요구사항 |
|----|----------|
| F2-1 | 재료 목록 + 옵션을 포함한 프롬프트를 구성하여 OpenRouter API 호출 |
| F2-2 | 모델: `nvidia/nemotron-3-super-120b-a12b:free` |
| F2-3 | 응답은 구조화된 JSON 형식으로 요청 |
| F2-4 | 스트리밍 응답(`stream: true`) 지원으로 실시간 텍스트 표시 |
| F2-5 | Rate Limit(429) 발생 시 자동 재시도 (최대 3회, 5초 간격) |

### F3. 레시피 카드 표시
| ID | 요구사항 |
|----|----------|
| F3-1 | 각 레시피를 카드 형태로 표시 |
| F3-2 | 카드 내용: 요리명, 예상 시간, 난이도, 사용 재료, 단계별 조리법 |
| F3-3 | 조리법은 번호가 매겨진 단계로 표시 |
| F3-4 | 사용된 재료는 보유 재료와 추가 필요 재료를 색상으로 구분 |
| F3-5 | 각 카드에 "저장하기" 버튼 제공 |
| F3-6 | "다시 생성하기" 버튼으로 전체 재생성 가능 |

## API 명세

### POST /api/generate-recipe

**Request**
```json
{
  "ingredients": ["달걀", "우유", "당근", "양파"],
  "options": {
    "servings": 2,
    "maxTime": 30,
    "diet": ["일반"],
    "count": 3
  }
}
```

**Response (성공)**
```json
{
  "recipes": [
    {
      "id": "uuid-1234",
      "title": "달걀 야채 볶음밥",
      "time": 20,
      "difficulty": "쉬움",
      "servings": 2,
      "usedIngredients": ["달걀", "당근", "양파"],
      "extraIngredients": ["밥", "간장"],
      "steps": [
        "당근과 양파를 잘게 썬다.",
        "팬에 기름을 두르고 야채를 볶는다.",
        "달걀을 풀어 넣고 밥과 함께 볶는다.",
        "간장으로 간을 맞춘다."
      ]
    }
  ]
}
```

**OpenRouter 호출 프롬프트 템플릿**
```
다음 재료로 만들 수 있는 레시피를 {count}개 추천해주세요.

재료: {ingredients}
인원: {servings}인분
요리 시간: {maxTime}분 이내
식단: {diet}

반드시 아래 JSON 형식으로만 응답하세요. 설명 없이 JSON만 출력하세요.
{
  "recipes": [
    {
      "title": "요리명",
      "time": 예상시간(숫자, 분),
      "difficulty": "쉬움|보통|어려움",
      "servings": 인원수(숫자),
      "usedIngredients": ["사용 재료 목록"],
      "extraIngredients": ["추가로 필요한 재료"],
      "steps": ["1단계", "2단계", ...]
    }
  ]
}
```

- 생성 중 스트리밍 텍스트 효과로 AI가 작성하는 느낌 표현
- 보유 재료는 초록색 체크, 추가 필요 재료는 주황색 표시
- 모바일에서 카드 스와이프로 레시피 탐색 가능

## 기술 스택

| 항목 | 선택 |
|------|------|
| 프레임워크 | Next.js 14 (App Router) |
| 스타일 | Tailwind CSS |
| 스트리밍 | Next.js Route Handler + ReadableStream |
| 상태 관리 | React useState / Context (Step 1 데이터 공유) |
| AI 모델 | `nvidia/nemotron-3-super-120b-a12b:free` via OpenRouter |

---

## 비기능 요구사항

- 스트리밍 첫 토큰 응답: 3초 이내
- 전체 레시피 생성 완료: 30초 이내
- JSON 파싱 실패 시 재시도 또는 원문 텍스트로 fallback 표시
- 생성된 레시피 데이터는 브라우저 sessionStorage에 임시 보관
